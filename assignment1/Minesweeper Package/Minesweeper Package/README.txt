                                Minesweeper                                
---------------------------------------------------------------------------

                                - Classes -                                

*-------------------------------------------------------------------------*
| Minesweeper.m  Main Minesweeper class                                   |
|                Type "Minesweeper" to lauch a GUI                        |
|                For more details, type "help Minesweeper"                |
*-------------------------------------------------------------------------*



                                - Scripts -                                

AItest.m       Script for programmatically testing a custom AI function



                               - Functions -                               

AIfcn.m        An example of a custom AI function



                               - MAT files -                               

data.mat       Contains a variety of data strucutres used by the GUI to
               render its in-game graphics. Don't modify/delete this file

scores.mat     Contains a single structure called HIGH_SCORES containing
               your best solution times. If this file doesn't exist, the
               leaderboard is cleared and a new scores.mat file will be
               created the next time you close a GUI



                              - Requirements -                             

Written using MATLAB R2011b (Mac) and MATLAB R2012a (Windows), but it is
forward and (probably) backward compatible

No toolboxes required
